## 📌 À propos du projet

**Admin_IC** est une application web d'administration conçue pour faciliter la gestion complète des réservations et des inscriptions aux événements. Elle offre aux administrateurs un tableau de bord centralisé pour suivre, valider et gérer les participants à leurs événements.

---

## ✅ Fonctionnalités

- 📝 **Gestion des inscriptions** : créer, modifier et supprimer des inscriptions aux événements
- 📅 **Gestion des réservations** : suivi en temps réel des réservations
- 👥 **Gestion des participants** : liste des inscrits avec informations détaillées
- 🔐 **Authentification sécurisée** : accès réservé aux administrateurs autorisés
- 📊 **Tableau de bord** : vue d'ensemble des événements, réservations et inscriptions
- 🔔 **Notifications** : alertes pour les nouvelles inscriptions et réservations

## 🚀 Installation

### Prérequis

Assurez-vous d'avoir installé :

- [Git](https://git-scm.com/)
- (Tout autre prérequis selon votre stack)

### Étapes

1. **Cloner le dépôt**

   ```bash
   git clone https://github.com/OnyxIkongo/Admin_IC.git
   ```

2. **Accéder au répertoire du projet**

   ```bash
   cd Admin_IC
   ```

3. **Installer les dépendances** *(si applicable)*

   ```bash
   # Exemple avec npm
   npm install

   # Exemple avec pip
   pip install -r requirements.txt
   ```

4. **Configurer les variables d'environnement** *(si applicable)*

   ```bash
   cp .env.example .env
   # Modifier .env avec vos paramètres
   ```

5. **Lancer l'application**

   ```bash
   # Exemple
   npm start
   ```

## 💻 Utilisation

1. Ouvrir l'application dans votre navigateur à l'adresse `http://localhost:3000` (ou le port configuré)
2. Se connecter avec vos identifiants administrateur
3. Utiliser le tableau de bord pour gérer les événements, réservations et inscriptions
